Privacy Configuration

Enable Cookie Consent Bar
If this is set to True, a cookie consent bar will be shown on the page alerting the user this site uses cookies. The user will need to acknowledge they agree.
Add Export Personal Data to My Account Privacy Panel
If set to True - Privacy will add a Download Personal Data button to the Privacy Panel under My Account. This will allow the user to download all personal data stored on the site.
Add Links to Privacy Polic / Terms Of Use to Registration Screen
If set to True, links to the site's Terms of Use and Privacy Policy will be displayed on the New User Registration Page.
Privacy Policy Static Page
Select the Static Page that contains the site Privacy Policy. If Do not Link to Privacy Policy is selected, no privacy policy link will be included on the registration page.
Terms of Use Static Page
Select the Static Page that contains the site Terms of Use statement. If Do not Link to Terms of Use is selected, no link will be included on the registration page.
Cookie Bar Style
.
Cookie Bar Position
Where to display the cookie bar.
Text For Cookie Bar
The custom text to show in the cookie bar that explains this site uses cookies.
Link to Learn More
If populated - a Learn More link will be displayed after the Cookie Bar Text. Generally this will be a link to your site's Privacy Policy.
